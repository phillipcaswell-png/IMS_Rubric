# IMS Charter

## Repository Constitutional Authority Pointer

Athena governance documents identify the IMS Charter as the supreme constitutional authority.

This repository pointer exists to make that authority reference discoverable and auditable from within the repository.

## Current Status

The full IMS Charter text is not currently present in this repository.

Until the full ratified Charter text is added, this document serves only as a discoverability pointer and SHALL NOT be interpreted as creating, amending, replacing, or expanding constitutional authority.

## Authority Rule

Where Athena documents reference the IMS Charter, they refer to the ratified constitutional authority governing Athena.

This pointer document does not supersede any ratified Charter text.

If the full IMS Charter is later added to the repository, that full ratified text SHALL become the canonical repository artifact and this pointer SHALL be replaced or revised accordingly.

## Audit Interpretation

Audits MAY cite this file only to establish that the repository contains an explicit reference path to the IMS Charter.

Audits SHALL NOT treat this pointer as sufficient evidence of the full Charter's contents.

If Charter text is required for a finding and the full Charter is unavailable, the correct evidence status is:

Insufficient Evidence.
